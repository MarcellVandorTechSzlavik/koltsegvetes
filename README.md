1. Projekt címe és egy rövid mondat
# Pro Költségvetés App
Egy böngészőben futó, modern pénzügyi tervező alkalmazás élő árfolyamokkal és grafikonokkal.

2. Főbb funkciók (Features)
Élő valutaárfolyamok: Automatikus frissítés (HUF, EUR, USD).

Intelligens adatbevitel: Megjegyzi a korábbi helyszíneket és bankokat a gyorsabb kitöltéshez.

Biztonsági korlát: Megakadályozza a negatív egyenlegbe való túlköltekezést.

Vizuális statisztika: Interaktív kördiagram a kiadások megoszlásáról (Chart.js).

Havi archiválás: Csak az aktuális hónap módosítható, a korábbiak fixálva maradnak.

Local Storage: Minden adat a böngésződben tárolódik, nincs szükség külső adatbázisra.

3. Hogyan használd? (How to use)
Töltsd le az index.html fájlt.

Nyisd meg bármelyik modern böngészőben.

Vegyél fel bevételt, majd rögzítsd a kiadásaidat.

4. Technikai részletek (Tech Stack)
HTML5 / CSS3 (Flexbox, CSS Variables)

Vanilla JavaScript (ES6+, LocalStorage, Fetch API)

Chart.js a grafikonokhoz.
